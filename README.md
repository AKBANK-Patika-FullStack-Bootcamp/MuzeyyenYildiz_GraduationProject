
# Site Management App
### Akbank FullStack Bootcamp Graduation project

## Features 
- ASP .NET CORE Web API
- MSSQL
- Swagger Ui
- React Js
- React Hooks
- Custom Hook
- React Router
- Redux
- React Bootstrap
- Axios
- Hunel React Credit Card (https://www.npmjs.com/package/reactjs-credit-card)
